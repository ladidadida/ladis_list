# ha-todo-manager

A Kanban-style todo manager add-on for Home Assistant, with assignees, recurring
tasks, and bidirectional Home Assistant automation integration.

This is a pre-implementation skeleton — see [AGENTS.md](AGENTS.md) for the full
design and [spec/roadmap.md](spec/roadmap.md) for build order. For installation,
once features land, see [DOCS.md](DOCS.md).
