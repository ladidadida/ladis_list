# Home Assistant Add-on Repository

A multi-add-on repository for [Home Assistant](https://www.home-assistant.io/) add-ons.

## Installation

1. In Home Assistant, go to **Settings → Add-ons → Add-on Store**.
2. Click the three-dot menu (⋮) → **Repositories** and add this repository's URL.
3. Install any of the add-ons listed below from the store.

## Add-ons

| Add-on | Description |
|---|---|
| [Shopping List](ha_shopping_list/DOCS.md) | A fast, mobile-friendly shopping list. |
| [Todo Manager](ha_todo_manager/DOCS.md) | A Kanban-style todo manager (pre-release, skeleton only — see its [roadmap](ha_todo_manager/spec/roadmap.md)). |
